# PRD: 팀 회고 웹앱 (Retro Board)

스프린트 회고를 온라인으로 진행할 수 있는 실시간 협업 보드.

## 배경

팀이 원격 근무 중이라 물리적 포스트잇 회고가 불가능하다. 기존 SaaS(EasyRetro, RetroTool)는 비싸고 커스터마이징이 안 된다. 우리 팀에 맞는 간단한 회고 도구를 직접 만든다.

## 제품 정의

### 한 줄 설명

실시간 협업 회고 보드. 카드 작성 → 투표 → 액션 아이템 도출.

### 핵심 기능

1. **회고 세션 생성**: 링크 공유로 참여 (로그인 불필요)
2. **3열 보드**: "좋았던 점", "개선할 점", "액션 아이템"
3. **익명 카드 작성**: 실시간으로 다른 참가자의 카드가 나타남
4. **투표**: 각 참가자가 카드당 최대 1표, 총 5표 제한
5. **타이머**: 진행자가 단계별 타이머 설정 (작성 5분, 투표 3분 등)
6. **결과 내보내기**: 마크다운으로 요약 다운로드

### 사용자

- 5~15명 규모 스크럼 팀
- 진행자(facilitator) 1명 + 참가자

## 비목표

- 사용자 계정/인증 시스템
- 회고 히스토리 저장 (세션 종료 시 데이터 삭제)
- 모바일 네이티브 앱
- 슬랙/지라 연동

## 성공 기준

| 지표 | 목표 |
|------|------|
| 동시 접속자 | 15명 이상 |
| 카드 반영 지연 | < 500ms |
| 세션 생성 ~ 첫 카드 작성 | < 30초 |
